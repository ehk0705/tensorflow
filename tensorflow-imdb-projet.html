Mini projet NLP avec TensorFlow et IMDB

Cette page montre un projet simple mais sérieux : classer des critiques de films en positives ou négatives avec TensorFlow et Keras.

Voir le code Voir la prédiction

Principe

Le jeu IMDB est un classique pour l’analyse de sentiments. Keras fournit ce jeu directement via `keras.datasets.imdb`, avec des séquences d’entiers déjà préparées. Les exemples Keras associés à IMDB limitent aussi souvent le vocabulaire et la longueur maximale des séquences. :contentReference[oaicite:1]{index=1}

1. Objectif du projet

On veut apprendre à un modèle à détecter si une critique de film exprime un avis positif ou négatif.

🎯

Tâche

Classification binaire : 0 ou 1.

📝

Entrée

Une séquence de mots représentant une critique.

📊

Sortie

Une probabilité indiquant si l’avis est positif.

2. Les données IMDB

Keras fournit IMDB directement par `keras.datasets.imdb.load_data()`. Les exemples officiels montrent aussi l’usage d’un vocabulaire tronqué, par exemple les 20 000 mots les plus fréquents, puis un bourrage des séquences à une longueur fixe comme 200. :contentReference[oaicite:2]{index=2}

Élément Rôle Exemple
Critique Texte du film « excellent film »
Séquence Version numérique [12, 45, 89, 3, ...]
Étiquette Sentiment 1 = positif, 0 = négatif

Dans cette version, on ne manipule pas le texte brut directement. On travaille avec des séquences déjà codées en entiers.

3. Préparation des données

Il faut limiter le vocabulaire, charger les données, puis forcer toutes les séquences à avoir la même longueur avec `pad_sequences`, comme dans les exemples Keras IMDB. :contentReference[oaicite:3]{index=3} 

import tensorflow as tf
from tensorflow import keras
from tensorflow.keras.preprocessing.sequence import pad_sequences

max_features = 20000   # vocabulaire limité aux 20 000 mots les plus fréquents
maxlen = 200           # longueur maximale de chaque critique

(x_train, y_train), (x_test, y_test) = keras.datasets.imdb.load_data(num_words=max_features)

x_train = pad_sequences(x_train, maxlen=maxlen)
x_test = pad_sequences(x_test, maxlen=maxlen)

print("Taille entraînement :", len(x_train))
print("Taille test :", len(x_test))
print("Forme d'une entrée :", x_train.shape)

Pourquoi limiter le vocabulaire ?

Pour réduire la taille du problème et garder les mots les plus utiles.

Pourquoi faire du padding ?

Parce qu’un réseau attend des entrées de forme régulière.

4. Modèle choisi

Ici, on utilise un modèle simple et pédagogique : Embedding + LSTM bidirectionnel + Dense. Keras propose bien un exemple officiel IMDB avec un LSTM bidirectionnel sur ce même jeu de données. :contentReference[oaicite:4]{index=4} 

from tensorflow import keras
from tensorflow.keras import layers

model = keras.Sequential([
    layers.Embedding(input_dim=max_features, output_dim=128, input_length=maxlen),
    layers.Bidirectional(layers.LSTM(64, return_sequences=True)),
    layers.Bidirectional(layers.LSTM(32)),
    layers.Dense(64, activation="relu"),
    layers.Dropout(0.3),
    layers.Dense(1, activation="sigmoid")
])

model.compile(
    optimizer="adam",
    loss="binary_crossentropy",
    metrics=["accuracy"]
)

model.summary()
🔢

Embedding

Transforme chaque entier en vecteur dense.

↔

Bidirectionnel

Lit la séquence dans les deux sens.

📌

Sigmoid

Donne une probabilité entre 0 et 1.

5. Entraînement

On entraîne avec une validation séparée. Keras couvre ce flux de travail complet, de la préparation au suivi des métriques. :contentReference[oaicite:5]{index=5} 

callbacks = [
    keras.callbacks.EarlyStopping(
        monitor="val_loss",
        patience=3,
        restore_best_weights=True
    )
]

history = model.fit(
    x_train,
    y_train,
    epochs=8,
    batch_size=64,
    validation_split=0.2,
    callbacks=callbacks
)

`EarlyStopping` évite souvent de continuer trop longtemps quand le modèle n’améliore plus la validation.

6. Évaluation finale

Une fois l’entraînement terminé, on mesure les performances sur le jeu de test. 

loss, accuracy = model.evaluate(x_test, y_test)
print("Perte test :", loss)
print("Précision test :", accuracy)

Ce que dit l’accuracy

Le pourcentage global de bonnes réponses.

Ce que dit la loss

Une mesure plus fine de l’erreur globale du modèle.

7. Exemple de prédiction

Le jeu IMDB fourni par Keras est pré-encodé. Donc, pour prédire sur une nouvelle phrase libre, il faut en pratique une étape supplémentaire de vectorisation adaptée. Cette page reste volontairement simple : on montre ici la prédiction sur une séquence déjà encodée du jeu de test. :contentReference[oaicite:6]{index=6} 

prediction = model.predict(x_test[:5])

for i, proba in enumerate(prediction):
    print("Exemple", i, "- probabilité positif :", float(proba[0]))

Si tu veux prédire directement sur du texte brut tapé à la main, il vaut mieux utiliser une page séparée avec `TextVectorization`.

9. Script complet de A à Z

Voici une version compacte, exécutable telle quelle. 

import tensorflow as tf
from tensorflow import keras
from tensorflow.keras import layers
from tensorflow.keras.preprocessing.sequence import pad_sequences

max_features = 20000
maxlen = 200

(x_train, y_train), (x_test, y_test) = keras.datasets.imdb.load_data(num_words=max_features)

x_train = pad_sequences(x_train, maxlen=maxlen)
x_test = pad_sequences(x_test, maxlen=maxlen)

model = keras.Sequential([
    layers.Embedding(input_dim=max_features, output_dim=128, input_length=maxlen),
    layers.Bidirectional(layers.LSTM(64, return_sequences=True)),
    layers.Bidirectional(layers.LSTM(32)),
    layers.Dense(64, activation="relu"),
    layers.Dropout(0.3),
    layers.Dense(1, activation="sigmoid")
])

model.compile(
    optimizer="adam",
    loss="binary_crossentropy",
    metrics=["accuracy"]
)

callbacks = [
    keras.callbacks.EarlyStopping(
        monitor="val_loss",
        patience=3,
        restore_best_weights=True
    )
]

model.fit(
    x_train,
    y_train,
    epochs=8,
    batch_size=64,
    validation_split=0.2,
    callbacks=callbacks
)

loss, accuracy = model.evaluate(x_test, y_test)
print("Perte test :", loss)
print("Précision test :", accuracy)

prediction = model.predict(x_test[:3])
print(prediction)

model.save("modele_imdb.keras")
