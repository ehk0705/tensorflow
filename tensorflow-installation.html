Comment utiliser TensorFlow : installation, configuration et exemples pédagogiques

Cette page explique pas à pas comment installer TensorFlow, préparer un environnement propre, vérifier que tout fonctionne, puis lancer de premiers exemples simples avec Keras.

Commencer Voir les exemples

Objectif

À la fin de cette page, tu sauras installer TensorFlow, créer un petit environnement de travail, exécuter un premier script, entraîner un petit modèle et comprendre l'enchaînement général.

1. Installation de TensorFlow

La méthode la plus simple et la plus propre consiste à utiliser Python, un environnement virtuel, puis une installation avec pip.

🐍

Étape 1 : installer Python

TensorFlow s'utilise avec Python. Il faut donc commencer par installer Python sur la machine.

📁

Étape 2 : créer un dossier de travail

Crée un dossier simple, par exemple tensorflow-demo, pour regrouper tes fichiers.

🧪

Étape 3 : créer un environnement virtuel

Cela évite les conflits entre bibliothèques et permet de garder un projet propre.

La méthode recommandée est d'utiliser venv et pip. C'est la voie officielle la plus simple pour débuter.

Commandes de base

python -m venv tf

# Activation sous Windows
tf\Scripts\activate

# Activation sous Linux ou macOS
source tf/bin/activate

pip install --upgrade pip

# Installation TensorFlow sur processeur
pip install tensorflow

Version processeur

C'est la solution la plus simple. Elle suffit largement pour apprendre, faire des essais et entraîner de petits modèles.

Version carte graphique

Si la machine possède une carte NVIDIA compatible, on peut viser une installation avec accélération. 

pip install tensorflow[and-cuda]

2. Vérifier que l'installation fonctionne

Après l'installation, il faut tester immédiatement. 

python -c "import tensorflow as tf; print(tf.__version__)"
python -c "import tensorflow as tf; print(tf.reduce_sum(tf.random.normal([1000, 1000])))"

Si tout va bien

Python affiche la version de TensorFlow, puis un résultat numérique. Cela signifie que TensorFlow fonctionne.

Pour vérifier la carte graphique

Si tu veux savoir si TensorFlow voit ton accélération GPU, utilise la commande ci-dessous. 

python -c "import tensorflow as tf; print(tf.config.list_physical_devices('GPU'))"

4. Comment utiliser TensorFlow au quotidien

L'utilisation générale suit presque toujours le même chemin.

1
Importer les bibliothèques

On commence par importer TensorFlow et Keras.

2
Préparer les données

Les données sont chargées puis mises dans un format exploitable.

3
Créer le modèle

On choisit les couches, la taille et la sortie souhaitée.

4
Compiler

On précise la perte, l'optimiseur et les métriques.

5
Entraîner

Le modèle apprend progressivement à partir des données.

6
Tester et utiliser

On évalue le modèle puis on l'utilise sur de nouvelles données.

5. Exemples pédagogiques

Voici plusieurs scripts simples pour apprendre progressivement.

Exemple 1

Premier test TensorFlow

Ce script vérifie simplement que TensorFlow fonctionne et manipule un tenseur. 

import tensorflow as tf

print("Version :", tf.__version__)

x = tf.constant([[1, 2], [3, 4]])
y = tf.constant([[5, 6], [7, 8]])

z = tf.matmul(x, y)
print(z)
Exemple 2

Petit réseau de neurones

Ici, on construit un modèle très simple avec Keras. 

import tensorflow as tf
from tensorflow import keras
from tensorflow.keras import layers

model = keras.Sequential([
    layers.Dense(16, activation="relu", input_shape=(4,)),
    layers.Dense(8, activation="relu"),
    layers.Dense(1, activation="sigmoid")
])

model.compile(
    optimizer="adam",
    loss="binary_crossentropy",
    metrics=["accuracy"]
)

model.summary()
Exemple 3

Entraînement sur de petites données

Ce petit exemple montre l'idée d'un apprentissage supervisé. 

import numpy as np
import tensorflow as tf
from tensorflow import keras
from tensorflow.keras import layers

x_train = np.array([
    [0, 0],
    [0, 1],
    [1, 0],
    [1, 1]
], dtype=float)

y_train = np.array([0, 1, 1, 1], dtype=float)

model = keras.Sequential([
    layers.Dense(4, activation="relu", input_shape=(2,)),
    layers.Dense(1, activation="sigmoid")
])

model.compile(
    optimizer="adam",
    loss="binary_crossentropy",
    metrics=["accuracy"]
)

model.fit(x_train, y_train, epochs=200, verbose=0)

predictions = model.predict(x_train)
print(predictions)
Exemple 4

Utiliser le jeu de données MNIST

C'est un classique pour débuter avec la reconnaissance de chiffres manuscrits. 

import tensorflow as tf
from tensorflow import keras
from tensorflow.keras import layers

(x_train, y_train), (x_test, y_test) = keras.datasets.mnist.load_data()

x_train = x_train / 255.0
x_test = x_test / 255.0

model = keras.Sequential([
    layers.Flatten(input_shape=(28, 28)),
    layers.Dense(128, activation="relu"),
    layers.Dropout(0.2),
    layers.Dense(10, activation="softmax")
])

model.compile(
    optimizer="adam",
    loss="sparse_categorical_crossentropy",
    metrics=["accuracy"]
)

model.fit(x_train, y_train, epochs=3)
model.evaluate(x_test, y_test)

6. Sauvegarder et recharger un modèle

Une fois le modèle entraîné, on peut l'enregistrer pour le réutiliser plus tard.

Enregistrer

model.save("mon_modele.keras")

Recharger

from tensorflow import keras

modele = keras.models.load_model("mon_modele.keras")

7. Erreurs fréquentes chez les débutants

Voici les problèmes les plus courants.

❌

Python non reconnu

Cela veut souvent dire que Python n'est pas installé correctement ou pas ajouté au chemin système.

⚠

Environnement non activé

TensorFlow peut sembler absent alors qu'il est bien installé, mais dans un autre environnement virtuel.

🧩

Version ou dépendance incompatible

Cela arrive quand plusieurs bibliothèques se mélangent. Un environnement propre règle souvent le problème.

Erreur classique

Installer beaucoup de paquets au hasard est une mauvaise idée. Il vaut mieux partir d'un environnement neuf.

Bonne méthode

Créer un nouvel environnement, mettre à jour pip, installer TensorFlow, tester immédiatement, puis seulement ajouter le reste.

8. TensorFlow, Keras et les LLM

Pour un débutant, il faut comprendre où se place TensorFlow dans l'ensemble.

⚙

TensorFlow

C'est la plateforme de calcul et d'entraînement des modèles.

🧱

Keras

C'est l'interface simple et pratique pour créer rapidement les modèles.

🧠

LLM

Les grands modèles de langage sont des modèles de deep learning beaucoup plus complexes, souvent entraînés sur des volumes immenses.

Vouloir commencer directement par un grand modèle de langage est une mauvaise approche. Il faut d'abord apprendre les bases avec de petits modèles.

9. Résumé final

Ce qu'il faut retenir

TensorFlow s'installe simplement avec Python, un environnement virtuel et pip. Keras est la meilleure porte d'entrée pour débuter. La bonne méthode est : installer, vérifier, écrire un petit script, entraîner un petit modèle, puis progresser.

Ordre conseillé

1. Installer Python
2. Créer un environnement virtuel
3. Installer TensorFlow
4. Vérifier l'installation
5. Tester Keras
6. Faire un premier modèle simple
7. Ensuite seulement aller vers des sujets plus avancés
